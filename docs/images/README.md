# Screenshot Checklist

Add real in-game screenshots here before using the addon suite in applications or portfolio pages.

Use these consistent names:

- `titan-bar.png`: the main VK Titan panel visible in-game.
- `bis-window.png`: the Best-in-Slot window with class/spec/phase controls visible.
- `session-xp.png`: session or XP module while playing.
- `utility-modules.png`: repair, gold, bag, or ammo modules visible together.

Before publishing screenshots, hide private chat, account names, BattleTag details, character names you do not want public, guild chat, and server/community information.
